`GET https://api.echovalue.dev/kv/<bucket>/<key>`

Retrieves the stored value for a key.

:::note
**Cost:** 1 credit
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Path parameters:**

| Parameter | Type | Description |
| --------- | ---- | ----------- |
| `bucket` | string | Logical namespace. Max 30 characters. |
| `key` | string | Key identifier. Max 30 characters. |

```sh
curl 'https://api.echovalue.dev/kv/default/mykey' \
  -H 'x-token: mytoken'
```

## Response

`200 OK` — The stored value as plain text.

```
hello world
```

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |
| `x-cost` | Credits consumed |

See [Response Headers](/response-headers/) for details.

## Status Codes

| Status | Meaning |
| ------ | ------- |
| `200` | Value returned |
| `400` | Invalid parameters |
| `401` | Invalid token |
| `402` | Insufficient credits |
| `404` | Key does not exist or has expired |

## Notes

- The response body is plain text, not JSON.
