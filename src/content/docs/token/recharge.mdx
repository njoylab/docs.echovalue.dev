`GET https://api.echovalue.dev/recharge`

Returns a payment link to add credits to your wallet.

:::note
**Cost:** free. This endpoint does not deduct wallet credits.
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Query parameters:**

| Parameter | Type | Description | Default |
| --------- | ---- | ----------- | ------- |
| `amount` | string | Recharge tier. Accepted values: `1` or `3`. | `1` |

## Response

`200 OK` — payment URL as plain text.

```
https://buy.stripe.com/<productID>?client_reference_id=<hash>
```

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |
| `x-cost` | Credits consumed (always 0) |

## Status Codes

| Status | Meaning |
| ------ | ------- |
| `200` | Recharge link returned |
| `401` | Invalid token |
| `405` | Method not allowed |

## Examples

```sh
curl 'https://api.echovalue.dev/recharge?amount=1' \
  -H 'x-token: mytoken'
```

Follow the returned URL to complete the payment.

The returned payment URL includes the wallet hash in `client_reference_id`.

:::note
The card transaction appears as **nJoyLab** on your statement.
:::
