`GET https://api.echovalue.dev/recharge`

Returns a Stripe payment link to add credits to your wallet.

:::note
**Cost:** free. This endpoint does not deduct wallet credits.
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Query parameters:**

| Parameter | Type | Description | Default |
| --------- | ---- | ----------- | ------- |
| `amount` | string | Recharge tier. Accepted values: `1` or `3`. | `1` |

## Response

`200 OK` — Stripe payment URL as plain text.

```
https://buy.stripe.com/<productID>?client_reference_id=<hash>
```

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |

## Status Codes

| Status | Meaning |
| ------ | ------- |
| `200` | Recharge link returned |
| `401` | Invalid token |
| `405` | Method not allowed |

```python

response = requests.get('https://api.echovalue.dev/recharge',
```

<TabItem label="PHP">

```php

$ch = curl_init('https://api.echovalue.dev/recharge?amount=1');
curl_setopt($ch, CURLOPT_HTTPHEADER, ['x-token: mytoken']);
```

<TabItem label="Go">

```go
package main

import (
    "fmt"
    "io"
    "net/http"
)

func main() {
    req, _ := http.NewRequest("GET", "https://api.echovalue.dev/recharge?amount=1", nil)
    req.Header.Set("x-token", "mytoken")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()
    body, _ := io.ReadAll(resp.Body)
    fmt.Println(string(body))
}
```

</TabItem>

Follow the returned URL to complete the payment.

The returned Stripe URL includes the wallet hash in `client_reference_id`.

:::note
The card transaction appears as **nJoyLab** on your statement.
:::
