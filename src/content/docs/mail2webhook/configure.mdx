`POST https://api.echovalue.dev/webhook`

Creates or updates a webhook for your wallet (upsert). Multiple webhooks can be created using different `id` values.

:::note
**Cost:** 1 credit
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |
| `Content-Type` | Must be `application/json` |

**Body** (`application/json`):

| Field | Type | Description | Required |
| ----- | ---- | ----------- | -------- |
| `id` | string | Unique identifier for this webhook (e.g. `slack`, `myserver`). Defaults to `0` if omitted. | No |
| `url` | string | Callback URL (must be HTTPS, publicly reachable) | Yes |
| `headers` | object | Custom headers to include in the callback request | No |
| `format` | string | Native message format: `slack`, `discord`, `teams`, `telegram`, `pagerduty`, `custom` | No |
| `template` | object | JSON template for `custom` format. Supports placeholders like `{{subject}}`, `{{from.name}}`, `{{attachments[0].filename}}`, fallback chains `{{text\|html}}`, and literal fallbacks `{{sender\|"unknown"}}`. Max 2048 bytes, 5 nesting levels. | No |
| `options` | object | Format-specific options (e.g. `chat_id` for Telegram, `routing_key` for PagerDuty). Max 10 keys, values up to 500 chars. | No |
| `includeAttachments` | boolean | Include email attachments in webhook payload (default: `true`). If `false`, attachments are stripped from the payload. | No |

For platform-specific setup guides see: [Slack](/mail2webhook-slack), [Discord](/mail2webhook-discord), [Teams](/mail2webhook-teams), [Telegram](/mail2webhook-telegram), [PagerDuty](/mail2webhook-pagerduty).

<TabItem label="Go">

```go
package main

import (
    "bytes"
    "encoding/json"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "id":     "slack",
        "url":    "https://hooks.slack.com/services/T00/B00/XXX",
        "format": "slack",
    }
    jsonData, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.echovalue.dev/webhook", bytes.NewBuffer(jsonData))
    req.Header.Set("x-token", "mytoken")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    client.Do(req)
}
```

## Response

`200 OK`

```
OK
```

| Status | Meaning |
| ------ | ------- |
| `200` | Webhook saved |
| `400` | Invalid URL or malformed request |
| `401` | Invalid token |
| `402` | Insufficient credits |

:::caution
The webhook URL must be HTTPS and publicly reachable. Localhost and private IP addresses are not accepted.
:::
