`POST https://api.echovalue.dev/webhook`

Creates a new webhook.

:::note
**Cost:** 1 credit.
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |
| `Content-Type` | Must be `application/json` |

**Body** (`application/json`):

| Field | Type | Description | Required |
| ----- | ---- | ----------- | -------- |
| `webhookId` | string | Public webhook identifier used for management endpoints. Defaults to `default` when omitted. | No |
| `name` | string | Human-friendly webhook name | No |
| `url` | string | Callback URL; must be HTTPS and publicly reachable | Yes |
| `headers` | object | Custom headers to include in the callback request | No |
| `format` | string | Native message format: `slack`, `discord`, `teams`, `telegram`, `pagerduty`, `custom` | No |
| `template` | object | JSON template for `custom` format. Max 2048 bytes, 5 nesting levels. | No |
| `options` | object | Format-specific options such as `chat_id` or `routing_key` | No |
| `includeAttachments` | boolean | Include email attachments in payloads. Default: `true`. | No |

The returned `email` address is opaque and may not match the chosen `webhookId`.

For format-specific setup see [Slack](/mail2webhook/formats/slack/), [Discord](/mail2webhook/formats/discord/), [Teams](/mail2webhook/formats/teams/), [Telegram](/mail2webhook/formats/telegram/), [PagerDuty](/mail2webhook/formats/pagerduty/), and [Custom](/mail2webhook/formats/custom/).

## Response

`200 OK` — JSON object.

```json
{
  "webhookId": "slack",
  "name": "slack",
  "webhook": "https://hooks.slack.com/services/T00/B00/XXX",
  "headers": {},
  "format": "slack",
  "options": {},
  "includeAttachments": true,
  "email": "r_abcd1234@hook.echovalue.dev",
  "hash": "a1b2c3d4e5f6..."
}
```

| Field | Type | Description |
| ----- | ---- | ----------- |
| `webhookId` | string | Public webhook identifier for management operations |
| `name` | string | Webhook name |
| `webhook` | string | Configured callback URL |
| `headers` | object | Custom headers |
| `format` | string | Native format, if configured |
| `template` | object | Custom template object when `format` is `custom` |
| `options` | object | Format options; sensitive values are redacted |
| `includeAttachments` | boolean | Whether attachments are included in webhook payloads |
| `email` | string | Opaque mailbox address that receives email for this webhook |
| `hash` | string | SHA256 hash of your token |

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |
| `x-cost` | Credits consumed |

See [Response Headers](/response-headers/) for details.

## Status Codes

| Status | Meaning |
| ------ | ------- |
| `200` | Webhook created |
| `400` | Invalid URL or malformed request |
| `401` | Invalid token |
| `402` | Insufficient credits |
| `500` | Internal server error |

<TabItem label="PHP">

```php
<?php
$data = json_encode([
  'webhookId' => 'slack',
  'name' => 'slack',
  'url' => 'https://hooks.slack.com/services/T00/B00/XXX',
  'format' => 'slack'

curl_setopt($ch, CURLOPT_POST, true);
curl_setopt($ch, CURLOPT_POSTFIELDS, $data);

  'x-token: mytoken',
  'Content-Type: application/json'
]);
curl_exec($ch);
curl_close($ch);
?>
```

<TabItem label="Go">

```go
package main

import (
    "bytes"
    "encoding/json"
    "net/http"
)

func main() {
    payload := map[string]interface{}{
        "webhookId": "slack",
        "name":   "slack",
        "url":    "https://hooks.slack.com/services/T00/B00/XXX",
        "format": "slack",
    }
    jsonData, _ := json.Marshal(payload)

    req, _ := http.NewRequest("POST", "https://api.echovalue.dev/webhook", bytes.NewBuffer(jsonData))
    req.Header.Set("x-token", "mytoken")
    req.Header.Set("Content-Type", "application/json")

    client := &http.Client{}
    client.Do(req)
}
```
