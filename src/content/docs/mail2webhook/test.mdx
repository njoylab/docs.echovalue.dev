`POST https://api.echovalue.dev/webhook/test`

Sends a dummy email payload to the specified webhook URL to verify it is working correctly.

:::note
**Cost:** 1 credit
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Query parameters:**

| Parameter | Type | Description | Required |
| --------- | ---- | ----------- | -------- |
| `mailboxId` | string | Opaque mailbox identifier to test | Yes |

package main

import (
    "encoding/json"
    "fmt"
    "io"
    "net/http"

func main() {
    req, _ := http.NewRequest("POST", "https://api.echovalue.dev/webhook/test?mailboxId=r_abcd1234", nil)
    req.Header.Set("x-token", "mytoken")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    var result map[string]interface{}
    json.Unmarshal(body, &result)
    fmt.Println(result)
}

## Response

`200 OK` — JSON object.

```json
{
  "success": true,
  "message": "test webhook delivered"
}
```

On failure:

```json
{
  "success": false,
  "message": "webhook call failed: context deadline exceeded (Client.Timeout exceeded while awaiting headers)"
}
```

| Field | Type | Description |
| ----- | ---- | ----------- |
| `success` | boolean | Whether the test webhook was successfully delivered |
| `message` | string | Confirmation or error message |

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |
| `x-cost` | Tokens consumed (1) |

See [Response Headers](/response-headers) for details.

| Status | Meaning |
| ------ | ------- |
| `200` | Test call processed |
| `401` | Invalid token |
| `402` | Insufficient credits |
| `405` | Method not POST |

:::note
The test sends a dummy payload with sample email data. Check your webhook endpoint logs to verify the full payload was received correctly.
:::
