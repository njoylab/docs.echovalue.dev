`DELETE https://api.echovalue.dev/webhook/<mailboxId>`

Removes a specific webhook from your wallet using its `mailboxId`.

:::note
**Cost:** 1 credit
:::

## Request

**Headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Path parameters:**

| Parameter | Type | Description | Required |
| --------- | ---- | ----------- | -------- |
| `mailboxId` | string | Opaque mailbox identifier to delete | Yes |

## Response

`200 OK`

```
OK
```

**Response headers:**

| Header | Description |
| ------ | ----------- |
| `x-balance` | Wallet balance after this call |
| `x-cost` | Tokens consumed (1) |

See [Response Headers](/response-headers) for details.

| Status | Meaning |
| ------ | ------- |
| `200` | Webhook deleted |
| `401` | Invalid token |
| `402` | Insufficient credits |
| `404` | No webhook configured |
