When `format` is set to `discord`, incoming emails are converted into Discord messages using the [Webhook API](https://discord.com/developers/docs/resources/webhook).

:::note
This format is available only for inbound email webhooks. Scheduled webhooks always send the scheduled-run JSON payload and do not use format adapters.
:::

## Get the Webhook URL

<Steps>

1. In Discord, right-click the channel and select **Edit Channel**.
2. Click **Integrations** → **Webhooks** → **New Webhook**.
3. Copy the generated webhook URL:
   ```
   https://discord.com/api/webhooks/000000000000000000/XXXXXXXXXXXXXXXXXXXXXXXXXXXX
   ```

</Steps>

## Configure in echoValue

```sh
curl 'https://api.echovalue.dev/webhook' \
  -H 'x-token: mytoken' \
  -H 'Content-Type: application/json' \
  -d '{
    "url": "https://discord.com/api/webhooks/000000000000000000/XXXX",
    "format": "discord"
  }'
```

## Output Preview

```
**Subject line**
From: Sender Name <sender@example.com>
Email body text
```

:::note
The plain text body is used. HTML is stripped automatically.
:::
