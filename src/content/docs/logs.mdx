## Retrieve Logs

`GET https://api.echovalue.dev/token/logs`

Returns a list of recent API calls made with your token. Logs have a TTL of 7 days and are deleted within 24 hours after expiration.

You can filter logs by endpoint type using the `type` parameter.

**Endpoint types:**

| Type | Description |
| ---- | ----------- |
| `kv` | Key-value operations (`GET`, `POST`, `DELETE` on `/kv/*`) |
| `webhook` | Webhook calls (`/webhook/call`) |
| `token` | Wallet info requests (`GET /token`) |
| `service` | Recharge operations |

**Request headers:**

| Header | Description |
| ------ | ----------- |
| `x-token` | Your API token |

**Query parameters:**

| Parameter | Type | Description | Default |
| --------- | ---- | ----------- | ------- |
| `n` | integer | Number of log entries to retrieve. Min: `1`. Default: `5`. | `5` |
| `type` | string | Filter logs by endpoint type. Values: `kv`, `webhook`, `token`, `manage`, `service`. If omitted, returns all logs. | — |

**Response:** `200` — JSON object.

```json
{
  "logs": [
    {
      "id": "jd9kImh8U4In3odfNeKf",
      "method": "GET",
      "path": "/default/mykey",
      "endpoint": "kv",
      "status_code": 200,
      "response_time_ms": 513,
      "error": "Key Not Found",
      "timestamp": "2023-12-07T12:14:16.124481Z",
      "expiration": "2023-12-14T12:14:16.12448Z",
      "cost": 1,
      "balance": 97
    }
  ],
  "n": 1
}
```

| Field | Type | Description |
| ----- | ---- | ----------- |
| `n` | integer | Number of entries actually returned |
| `logs` | array | Array of log entries |

**Log entry fields:**

| Field | Type | Description |
| ----- | ---- | ----------- |
| `id` | string | Unique log entry ID |
| `method` | string | HTTP method used (`GET`, `POST`, `DELETE`) |
| `path` | string | Full request path including query parameters |
| `endpoint` | string | API category (`kv`, `token`, `webhook`) |
| `status_code` | integer | HTTP status code returned by the request |
| `response_time_ms` | integer | Response time in milliseconds for the request |
| `error` | string | Error message, if any (omitted on success) |
| `timestamp` | string (ISO 8601) | When the request was made |
| `expiration` | string (ISO 8601) | When this log entry expires (7 days after creation) |
| `cost` | integer | Credits deducted for that request |
| `balance` | integer | Wallet balance after that request |
**HTTP status codes:**

| Status | Meaning |
| ------ | ------- |
| `200` | Logs returned |
| `401` | Invalid token |

Note: fetching logs is free and does not deduct credits from your wallet. The service will return at most 100 logs per request. If you request more than 100 logs, the response will be truncated to 100.

### Filtering by Endpoint Type

Use the `type` parameter to filter logs by endpoint:

```sh
# Get only KV operations
curl 'https://api.echovalue.dev/token/logs?type=kv' -H 'x-token: mytoken'

# Get only webhook calls
curl 'https://api.echovalue.dev/token/logs?type=webhook&n=10' -H 'x-token: mytoken'
```

<TabItem label="Go">

```go
package main

import (
    "encoding/json"
    "fmt"
    "io"
    "net/http"
)

func main() {
    // Get all logs
    req, _ := http.NewRequest("GET", "https://api.echovalue.dev/token/logs?n=5", nil)
    req.Header.Set("x-token", "mytoken")

    client := &http.Client{}
    resp, _ := client.Do(req)
    defer resp.Body.Close()

    body, _ := io.ReadAll(resp.Body)
    var result map[string]interface{}
    json.Unmarshal(body, &result)

    if logs, ok := result["logs"].([]interface{}); ok {
        fmt.Printf("Returned %v logs\n", result["n"])
        for _, li := range logs {
            l := li.(map[string]interface{})
            fmt.Printf("%v %v %v status=%v time=%vms cost=%v balance=%v id=%v\n",
                l["timestamp"], l["method"], l["endpoint"], l["status_code"], l["response_time_ms"], l["cost"], l["balance"], l["id"])
        }
    }

    // Get only KV logs
    req2, _ := http.NewRequest("GET", "https://api.echovalue.dev/token/logs?type=kv", nil)
    req2.Header.Set("x-token", "mytoken")
    resp2, _ := client.Do(req2)
    defer resp2.Body.Close()
}
```

:::note
Log reads are free. By default the API returns 5 logs when `n` is not provided. The maximum number of logs you can request in a single call is 100.

Note that logs may have a delay of a few seconds from when they are written to when they become readable.
:::
